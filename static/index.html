Cloudflare 配置说明

1 获取 API Token

请到 Cloudflare 控制台 "我的个人资料" → "API令牌" 创建包含以下权限的 API 令牌：

  • 账户: Cloudflare Tunnel [编辑]
  • 账户: Cloudflare Tunnel [读取]
  • 区域: DNS [编辑]
  • 区域: DNS [读取]

点击 "继续以显示令牌" 后，复制获得的 API Token

2 获取 Account ID

在 Cloudflare 控制台首页，右侧栏即可看到 "账户ID"，点击复制即可
