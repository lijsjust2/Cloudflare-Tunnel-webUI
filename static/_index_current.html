Cloudflare 配置说明

在上侧填写 Cloudflare 账号 ID 与 API 令牌，用于创建、查看与连接 Tunnel。

1、获取 API Token
  1. 前往 Cloudflare 控制台的 「配置文件」→「API 令牌」 页面
  2. 创建 API 令牌，需包含以下权限：
  • 账户: Cloudflare Tunnel [编辑]
  • 账户: Cloudflare Tunnel [读取]
  • 区域: DNS [编辑]
  • 区域: DNS [读取]

点击本页面的「保存」按钮后，即可获得并存储 API Token。

2、获取 Account ID

在 Cloudflare 控制台任意页面，按 Ctrl+K 或 ⌘+K 打开搜索框，输入 ID，建议命令中会出现「复制账户 ID」选项，点击即可复制。
